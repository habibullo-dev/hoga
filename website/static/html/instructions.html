Welcome!

Meet your ultimate personal space. Here's a quick step-by-step guide on how to use the app.

Control Panel

Hover to the top of the page to initialize the control panel. The control panel displays all the available widgets and customizable settings.

Interactive Widgets

Select the edit button and pick any of the available widgets you want to add.

Generate Layouts

Select any of the preset themes or try some of the developer picks for some cool features!

Save Themes

Click on the save button to automatically restore your settings for the next visit.
